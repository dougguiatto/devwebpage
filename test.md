Create a modern, high-converting landing page for a tech consulting company called DEV4LAB, specialized in myLIMS implementation and development services, using just css, html and javascript.

## 🎯 GOAL
The goal of this landing page is to generate leads (contact via WhatsApp or form) from companies in the laboratory sector that use or want to implement myLIMS.

---

## 🧠 CONTEXT ABOUT THE COMPANY
DEV4LAB is a startup specialized in:
- Implementation of myLIMS
- Development of custom reports (Power BI, Report Builder)
- Project management for lab systems

The company focuses on:
- Speed
- Technical expertise
- Agile delivery
- Deep knowledge of myLIMS

---

## 🎨 BRAND GUIDELINES (VERY IMPORTANT)

Follow strictly the brand manual:

Primary Colors:
- Dark Blue: #1D1A3B
- Orange: #F5A135
- Light Gray: #F2F2F2
- White: #FFFFFF

Style:
- Clean, modern, minimal
- High contrast between sections
- Use orange for CTAs and highlights
- Use dark blue for text and strong sections

Typography:
- Similar to "New Order" (modern geometric sans-serif)
- Bold headlines, clean body text

Visual Identity:
- Tech + laboratory feeling
- Subtle abstract shapes (circles, lab references)
- Use illustrations instead of stock photos if possible

## 💡 UX / UI REQUIREMENTS

- Mobile-first responsive
- Smooth scroll between sections

- Soft shadows on cards
- Rounded corners (8px–12px)
- Subtle hover effects
- Fast loading

## 🚫 AVOID

- Cluttered layout
- Too much text
- Generic stock images
- Weak CTA visibility
- Inconsistent spacing
